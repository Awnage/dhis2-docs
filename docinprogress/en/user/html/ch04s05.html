4.5. Assign names to organisation unit levels

You can specify a contextual name for each level in the hierarchy, for example "Country", "Province", "District" or "Health Facility". The contextual names are used all over the application where levels are referred to. This page will take some time to load if the organisation unit hierarchy is very big.

  1. Click Apps > Maintenance.

  2. Click Organisation units.

  3. Select the organisation units you want to move:

    1. In the hierarchy tree to the left, select the organisation unit you want to move.

    2. In the hierarchy tree to the right, select where you want to move the organisation unit to.

    3. Repeat selection for all the organisation units you want to move.

  4. Click Move x organisation units, where x stands for the number of organisation units you have selected.

    Your changes are immediately reflected in the left side hierarchy tree.
