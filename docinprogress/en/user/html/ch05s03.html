5.3.1. Edit a data element

  1. Click Apps > Maintenance > Data elements > Data Element.

  2. In the list, find the data element you want to modify.

  3. Click the options menu and select Edit.

  4. Modify the options you want.

  5. Click Save.

5.3.2. Clone an existing data element

Once a data element is cloned, you can simply edit the details which need to be changed and then press "Save".

Cloning a data elements can save time when you create many similar data elements which differ in only a few details.

  1. Click Apps > Maintenance > Data elements > Data Element.

  2. In the list, find the data element you want to modify.

  3. Click the options menu and select Clone.

  4. Modify the options you want.

  5. Click Save.

5.3.3. Modify sharing settings for a data element

  1. Click Apps > Maintenance > Data elements > Data Element.

  2. In the list, find the data element you want to modify.

  3. Click the options menu and select Share.

  4. Modify the options you want.

  5. Click Close.

5.3.4. Delete a data element

You can only delete a data element if no data is attached to the data element itself.

  1. Click Apps > Maintenance > Data elements > Data Element.

  2. In the list, find the data element you want to delete.

  3. Click the options menu and select Delete.

  4. Click OK to confirm.

5.3.5. Display data element details

  1. Click Apps > Maintenance > Data elements > Data Element.

  2. In the list, find the data element you want to view.

  3. To display all metadata about a the selected data element, click the options menu and select Show details.

5.3.6. Translate a data element

  1. Click Apps > Maintenance > Data elements > Data Element.

  2. In the list, find the data element you want to translate.

  3. Click the options menu and select Translate.

  4. Select a locale.

  5. Enter Name, Short name, Description and Form name.

  6. Click Close.
